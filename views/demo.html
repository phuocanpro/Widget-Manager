How to Add Widget to Your Website

Copy this code and paste it before the closing </body> tag of your website:

<!-- Widget Manager Chat Widget --> <script> (function() { var script = document.createElement('script'); script.src = 'http://localhost:3003/widget/1.js'; script.async = true; document.head.appendChild(script); })(); </script> <!-- End Widget Manager Chat Widget -->

Replace the widget ID (1) with your actual widget ID from the dashboard.
